## 📋 Understanding the features

The application manages several complex tasks. Here is what each feature does for your search ranking.

### AI Search Optimization
The program automatically updates your site metadata. It ensures Google AI Overviews and other tools see your content as authoritative. This happens in the background. You do not need to edit your website code manually.

### LLMs.txt generation
A llms.txt file acts as a map for AI crawlers. It tells them exactly which pages to read and which pages to ignore. This tool creates and updates this file automatically. It prevents errors that lead to bad search results.

### MCP support
MCP stands for Model Context Protocol. It allows AI agents to interact with your data in real-time. By enabling MCP, your site gains the ability to talk back to AI models. This increases the chances that an AI agent references your data during a conversation with a user.

### OAuth integration
OAuth adds a layer of security to your data connections. It ensures that only authorized AI agents pull your information. The application handles the creation of these secure tokens so you do not have to handle certificates or passwords yourself.
